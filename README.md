# My-Portfolio

A modern, professional, and interactive portfolio website built with React and Tailwind CSS.

## âœ¨ Features
- **Dark theme** with yellow accent and animated gradient background
- **Responsive design** for desktop and mobile
- **Collapsible sidebar** on mobile, fixed sidebar on desktop
- **Animated section entrances** (fade-up on scroll)
- **3D tilt effect** on project cards
- **Light yellow border on hover** for main containers (About, Skills, Projects, Contact, Education, Certifications)
- **Animated button effects** (scale, shadow, and ripple on click)
- **Project modals** for full descriptions
- **Fun facts** and social links in sidebar
- **Glassmorphism-inspired cards** (optional, easy to enable)
- **No text selection marker** for a clean, app-like feel

## Sections
- Sidebar (with fun facts, social links, and avatar)
- About Me & What I'm Doing
- Education & Certifications (vertical timeline)
- Skills (grouped grid)
- Projects (with modals, icons, and statistics)
- Contact (animated form)

## Getting Started

1. **Install dependencies:**
   ```sh
   npm install
   ```
2. **Start the development server:**
   ```sh
   npm start
   ```
3. **Build for production:**
   ```sh
   npm run build
   ```

## Tech Stack
- React
- Tailwind CSS
- AOS (Animate On Scroll)

## Animations & Effects
- Section fade-in on scroll (AOS)
- Animated gradient background
- Button hover, scale, and ripple effects
- 3D tilt on project cards
- Animated yellow borders on hover

## Customization
- Edit content in `src/components/`
- Update images in `src/assets/`
- Tweak styles in `src/index.css` and `src/components/index.css`
